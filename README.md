# scavenger-hunt-kmp

## Overview
Goal is to create a mobile application utilizing the power of Kotlin Multiplatform; additionally, harnessing the power of AI with Gemini.
The inspiration of this side project is drawn from another [AI-powered Flutter Project](https://github.com/mkobuolys/ai-photo-scavenger-hunt) that also has a similar game mechanics and hopefully to add some more features down the road.

## Stack
- Jetpack Compose (Android)
- SwiftUI (iOS)
- Koin
- Ktor Client
- AndroidX ViewModel
